# 기획 개요

## 기획 배경

ADHD 당사자로서 장기적 할 일을 관리하는 데 어려움을 겪어 만들게 되었다.

할 일을 관리하기 어려운 사람을 위한 주간 계획 위주 할 일 관리 서비스이다.

## 문제 정의

ADHD 환자는 다음과 같은 문제를 겪고 있다.

1. 우선 순위 파악의 어려움
    1. 무엇이 중요한 일인지 이해하지 못함.
2. 시간 관념의 왜곡
    1. 할 일을 하는 데 걸리는 시간을 과소평가하거나 과대평가함.
    2. 할 일을 ‘지금 해야 한다 vs 하지 않아도 된다’라고 이분법적으로 판단.
3. 실행 기능 약함
    1. 
    2. 주의력 결핍으로 인해 과제를 끝까지 수행하기 어려움.

## 해결책 제시

CBT(인지행동치료) 기반 솔루션.

1. 우선 순위 제시
    1. 일일 포괄 실행 목록을 통해 하루에 반드시 완료해야 하는 할 일이 무엇인지 목록화.
    2. 시급도와 영향력을 시각화 및 정렬.
2. 시간 시각화
    1. 시간을 블록으로 만듦.
    2. 할 일과 시간을 연결지어 생각하도록 함.
3. 게이미피케이션
    1. 일정 타일을 블록으로 드래그&드롭하게 함으로써 시간 관리를 게임처럼 즐겁게 여기도록 함.
    2. 좋아하는 일에는 주의력을 소모하지 않기에 부담 경감.

# 기능 명세

## 용어 정리

주간 할 일 카드 : 주간 단위로 작성하는 할 일. 카드 형태의 리스트로 볼 수 있으며, 하위 태스크를 가질 수 있음.

스프린트 할 일 : 주간 할 일 카드 중 마감 기한이 이번 주까지인 카드. 향후 사용자가 자유롭게 편집할 수 있도록 할 예정임.

할 일 블록 : 사용자가 계획표에 넣을 수 있는 최소 할 일 단위. 레고 블록 같은 개념으로, 개수를 조절할 수 있음.

일정 타일 : 위클리 캘린더에서 시각적으로 표시되는 타일. 실제 사용자의 일정을 나타냄.

포괄 실행 목록 : 인지행동치료에서 사용되는 용어로, 해당 서비스에서는 ‘오늘 꼭 해야 하는 세 가지 할 일’이라는 의미로 쓰임.

## 흐름에 따른 기능 정리

### 로그인 및 회원가입

로그인 페이지에서 회원가입을 할 수 있다.

구글 API 기반 로그인만 지원한다.

### 주간 할 일

사용자는 주간 할 일 페이지에서나 주간 계획 화면의 주간 할 일 탭을 활용해 주간 할 일을 추가할 수 있다.

일정을 추가할 때는 할 일의 이름과 마감 기한과 영향력을 설정해야 한다. 카테고리는 선택적으로 기입할 수 있다. 진행도는 기본적으로 0으로 주어지며, 향후 수정할 수 있다.

카테고리를 선택하지 않으면 ‘미정’ 카테고리로 분류된다.

카테고리로 필터링이 가능하며, 카테고리는 색상을 지정해 관리할 수 있다. 초기엔 카테고리 색상은 정해진 디폴트 색상만 사용 가능하게 할 예정으로, 가짓수는 총 5개를 생각 중이다. 이후 테마나 사용자 지정 색상 기능을 추가할 예정.

마감 기한과 영향력에 따라 할 일이 자동 정렬된다.

할 일의 시급성은 몹시 시급, 시급함, 여유로움 세 개로 분류된다. 각각 카드의 색이 붉은색, 노란색, 초록색으로 분류된다.

### 주간 계획

위클리 캘린더 형태의 화면으로, 아래에 주간 할 일 보기 탭을 열고 닫을 수 있다. 이때 주간 할 일에서의 ‘스프린트 할 일’을 불러오게 된다. 불러올 때 최소 단위는 하위 태스크이다. 스프린트 할 일은 기본적으로 이번 주 할 일이나, 이번 주의 할 일이 아니더라도 추가할 수 있다.

주간 할 일 보기 탭에서 주간 할 일 블록을 드래그/드롭으로 끌어다 주간 계획에 놓을 수 있다. 이때 

주간 계획의 매 요일마다 상단에 일일 포괄 목록이 나타난다. 일일 포괄 실행 목록은 3개까지만 배정할 수 있다.

오늘 해야 하는 포괄 실행 목록을 총 세 개까지 작성할 수 있다. 일정 외에 가장 중요한 할 일을 정리하는 기능이다.

### 일간 계획 (향후 추가 예정)

하루 계획을 작성할 수 있다.

할 일 일정 블록으로 채울 수 있다.

일일 포괄 실행 목록을 수정할 수 있다.

### 기록 (향후 추가 예정)

통계 요약

### 구글 캘린더 연동 (향후 추가 예정)

버튼을 누르면 주간 계획 일정과 할당된 할 일을 구글 캘린더와 연동한다.

향후 자동 동기화 기능 추가 예정.

### 시간표 (향후 추가 예정)

매번 반복되는 일정을 시간표로 지정 가능.

시간표가 적용되는 기간을 설정할 수 있음. 새로운 시간표 생성 가능.
